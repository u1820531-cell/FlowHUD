## FlowHUD
A flow-state execution dashboard that measures
focus quality, not just time spent.

### Why
Most productivity tools track minutes.
FlowHUD tracks execution quality using:
- Flow Score™
- Streak logic
- Pause detection
- Adaptive recovery timing

### Features
- Flow-state Pomodoro system
- Subject → chapter execution tracking
- Flow Score™ performance rating
- Weekly focus analytics
- Adaptive break logic
- Immersive soundscape
- Offline-first (localStorage)

### Tech
Vanilla JS, Canvas API, LocalStorage, CSS Glassmorphism
